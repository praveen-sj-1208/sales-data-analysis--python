# Sales Data Analysis Using Python

## Project Overview
This project analyzes sales data using Python to understand revenue trends, product performance,
and category-wise sales insights.

## Tools Used
- Python
- Google Colab
- Pandas
- GitHub

## Key Insights
- Total revenue and order volume were calculated successfully.
- Product-wise and category-wise performance was analyzed.
- Monthly sales trends were identified to support business decisions.

## Files Included
- Week2_Sales_Data_Analysis.ipynb
- week2_sales_analysis_output.csv
